== Description ==

Wanda Lite is a child theme of Sheeba Lite WordPress theme, optimized to work with WooCommerce e-commerce plugin. Wanda Lite WordPress theme offers an optional header image and header banner, new grid layout on homepage, new layout for post comments and new Google Fonts.
Created by ThemeinProgress, http://www.themeinprogress.com
